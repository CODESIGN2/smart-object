# How to use

clone repo

```
cd smart-object/test/gimp-plugin
bash install.sh
gimp
```

> **Notice:** This was only tested on Ubuntu machines running xenial with GIMP 2.8. It's not a stable script it's an example of very basic integrating with GIMP
